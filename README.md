Santiago Arredondo Torres — Profile Website

Brief description

This repository contains a small, secure, easy-to-maintain static profile website for Santiago Arredondo Torres (Software Engineer). The site will be minimal, accessible, responsive, and simple to update.

Project goals

- Present a clear, professional profile (About, Skills, Projects, Experience, Contact, Resume)
- Keep dependencies minimal; prefer a static site or static site generator
- Prioritize accessibility and performance
- Make the site easy to maintain and update with plain HTML/CSS/MD or a lightweight generator

Repository structure (initial)

- /project/  — design assets, images, and working files
- README.md  — this file
- tasks.md   — task list and planned work
- tasks-progress.md — progress tracking template
- scope.md   — project scope and acceptance criteria

How to preview locally

- If the site is static HTML, a simple local server works: `python -m http.server` from the site root or any static file server.
